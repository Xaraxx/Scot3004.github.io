Instalar ArchBang

Jun 7, 2011 Sergio C. Orozco Torres

Como todos bien sabemos en linux existen distribuciones para todo tipo de personas, desde los que quieren hacerlo todo hasta los que lo quieren todo facil
bueno yo combino ambos quiero hacer todo de la forma facil
por que ArchBang si existen otras distribuciones linux mas sencillas, mas comunes, donde tu lo haces todo. La razon por la que escoji archbang es por que en realidad es ArchLinux preparado para usar con el escritorio openbox, y es que seamos realistas, existen equipos donde al instalar archlinux en su estado puro podemos perdernos y morir en el intento, claro ArchLinux RLZ!!!!

Procedimiento
Lo primero que necesitamos para instalar archbang es descargar la iso: http://archbang.org/download
podemos usar la guia oficial (en ingles)
http://wiki.archbang.org/index.php?title=Installation
o usar la mia
Recomiendo preparar las particiones en windows en caso que sea el que tenemos preparado
http://windows.microsoft.com/es-XL/windows-vista/Create-and-format-a-hard-disk-partition
dejando una particion vacia con el espacio que le queramos dedicar a Linux.
Preparamos una memoria usb, si queremos podemos usar una memoria SD o cualquier dispositivo que pueda arrancar nuestro equipo.
es necesario que este en formato fat o fat32 para que funcione no he podido lograrlo con NTFS
si estamos en windows usamos universal usb installer, nos da la opcion de descargar la iso o escojer una que ya tenemos descargada
Sino podemos quemar un cd que contenga ArchBang y nos ahorramos el paso de hacerlo booteable

Iniciamos ArchBang desde el medio que escojimos
























Al principio veremos una pantalla como esta






Instalamos Archbang usando:
sudo abinstall, los pasos de instalacion lo encuentran en la guia oficial de archbang
http://wiki.archbang.org/index.php?title=Installation
podemos usar la de ArchLinux
https://wiki.archlinux.org/index.php/Gu%C3%ADa_Oficial_de_Instalaci%C3%B3n
si somos principiantes usamos la guia para principiante de archlinux:
https://wiki.archlinux.org/index.php/Gu%C3%ADa_para_Principiantes

antes de continuar quiero recomendar la lectura de la guia de uso de pacman:
https://wiki.archlinux.org/index.php/Pacman_(Espa%C3%B1ol)

Recomiendo actualizar el sistema, podemos hacerlo tambien una vez este instalado

# pacman -Syu

sudo pacman -Syu, podemos usar este comando cada vez que queramos tener nuestro sistema al dia

archbang trae como reproductor mplayer, ademas de tener codecs de mp3 instalados
lo que necesitamos es instalar java si queremos programar o ejecutar applets hecho en el mismo:
si usamos openjdk instalamos icedtea para los applets

# pacman -S openjdk6 icedtea-web

https://wiki.archlinux.org/index.php/Java_(Espa%C3%B1ol)
https://wiki.archlinux.org/index.php/Browser_Plugins

ahora explicare algunos programas de frecuente uso
mplayer reproductor por consola, reproduce videos, importantisimo http://www.mplayerhq.hu/design7/news.html
gmplayer, GUI para mplayer
si queremos instalamos vlc

# pacman -Sy vlc

http://www.videolan.org/vlc/
Gimp editor de imagenes avanzado
http://www.gimp.org/
Gmrun lanzador de aplicaciones alt+f2
http://sourceforge.net/projects/gmrun/
http://fausto23.wordpress.com/2009/02/22/gmrun-un-lanzador-alternativo/
lxterminal y lxappaerance de lxde
http://lxde.org/
uno es el emulador del terminal el otro para cambiar la apariencia del espacio de trabajo
Conky: gadgets del escritorio
http://conky.sourceforge.net/
Abiword, Gnumeric parte de gnome office
http://live.gnome.org/GnomeOffice
packer, para instalar paquetes de aur
oblogout cuadro para cerrar sesion y apagar
entre otras aplicaciones
