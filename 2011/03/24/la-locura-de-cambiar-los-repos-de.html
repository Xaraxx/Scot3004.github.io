La Locura de Cambiar los repos de Chakra por los de Arch

Mar 24, 2011 • Sergio C. Orozco Torres

empezando advierto que esto no se debería hacer
lo hice por que me puse a jugar con yaourt
para tenerlo mas reciente instale pacman bajandome el paquete y tirando
scot3004 ~/Downloads $ sudo pacman -U pacman-3.5.1-1-i686.pkg.tar.gz
luego actualize la base de datos con el comando que me decia
pero me di cuenta de algo. el repo platform no era compatible, y pude decir que mal....
porque contiene paquetes importantisimos y algunos mas recientes que los de arch, en fin
finalmente decidi cambiar todo o nada
y deje los repos de arch de primeros

editando /etc/pacman.conf asi:

#-------------------------------------------------------------------
#Repo Kde Chakra Project
[desktop]
Server = http://chakra-project.org/repo/desktop/i686/

# Useful and beautiful plasmoids and themes for KDE4.
[kde4-eyecandy-32]
Server = http://archlinuxgr.tiven.org/kde4-eyecandy/i686/
#--------------------------------------------------------------------


#Repo Archlinuxfr
[archlinuxfr]
Server = http://repo.archlinux.fr/i686

#----------------------------------------------------------------------
#Repo Archlinux
[core]
Include = /etc/pacman.d/mirrorlistArch
#Server = http://chakra-project.org/repo/core/i686/
#Server = http://archlinux.puzzle.ch/core/os/i686/

[extra]
Include = /etc/pacman.d/mirrorlistArch
#Server = http://archlinux.puzzle.ch/extra/os/i686/


[community]
Include = /etc/pacman.d/mirrorlistArch
#Server = http://archlinux.puzzle.ch/community/os/i686/

#--------------------------------------------------------------------
#repos exclusivos Chakra
#Repo Archlinux
[core]
Include = /etc/pacman.d/mirrorlist

[apps]
Include = /etc/pacman.d/mirrorlist

[platform]
Include = /etc/pacman.d/mirrorlist

[games]
Include = /etc/pacman.d/mirrorlist

#----------------------------------------------------------------------
y los mirrors escojidos
/etc/pacman.d/mirrorlist
Server = http://chakra-project.org/repo/$repo/i686

/etc/pacman.d/mirrorlistArch
Server = http://mirrors.kernel.org/archlinux/$repo/os/i686

luego
#pacman -Syu
